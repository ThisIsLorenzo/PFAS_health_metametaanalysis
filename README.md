# Human health impacts of PFAS exposure: a second-order meta-analysis 
## Study Overview
This repository contains all data, code, and analysis materials for the research study: "*Human health impacts of PFAS exposure: a second-order meta-analysis*".

Our study addresses a critical global health challenge posed by per- and polyfluoroalkyl substances (PFAS), a class of synthetic chemicals with ubiquitous human exposure and mounting evidence of toxicity.

## Repository Structure
### Analysis scripts (`/R`)
Contains R markdown files (`.Rmd`) with corresponding HTML outputs documenting the analysis.
-  `/R/analysis_code_mma.Rmd`: General analysis code.
-  -  `/R/analysis_code_mma.html`: HTML file (visualisation) of general analysis code.
### Data files (`/data`)
Houses all datasets in CSV format used in the analysis.
-  `/data/ma_details.csv`: Study Identification Information
-  `/data/ma_e_details.csv`: Estimate Identification Information (model outputs from the included meta-analyses)			
-  `/data/amstar2.csv`: Results of the AMSTAR 2 appraisal tool
### Visualisations (`/figs`)
Includes all figures generated for the manuscript and supplementary materials in PNG format.
### Project file
`PFAS_health_metametaanalysis` - RStudio project file for convenient access to the complete analysis environment.
